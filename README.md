# Бильярд 2D

Простая браузерная игра в бильярд на HTML5 Canvas.

## Как запустить
1. Откройте файл `index.html` в браузере (Chrome).
2. Выберите режим и нажмите "Начать игру".

## Возможности
- 2 режима: 2 игрока и против компьютера
- Простая физика и управление мышью
- Звуки и эффекты

## Структура
- `index.html` — разметка и Canvas
- `style.css` — стили
- `script.js` — логика игры
- `/assets` — изображения и звуки
